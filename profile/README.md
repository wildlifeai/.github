# Welcome to Wildlife.ai 🌿🤖

We are a New Zealand-based charity that uses artificial intelligence to accelerate wildlife conservation.

Below is an overview of our active project ecosystems and their associated repositories.

---

## 📱 Wildlife Watcher Ecosystem
Our flagship platform for managing and deploying smart camera traps. It allows researchers to configure hardware in the field and sync data seamlessly.

* **[Mobile App](https://github.com/wildlifeai/wildlife-watcher-mobile-app):** The primary React Native application for field researchers to connect to and configure hardware via Bluetooth.
* **[Backend](https://github.com/wildlifeai/wildlife-watcher-backend):** Core database logic and authentication powered by Supabase and PostgreSQL.
* **[Web Portal](https://github.com/wildlifeai/wildlife-watcher-web-portal):** A browser-based dashboard for managing large-scale deployments and data visualisation.
* **[User Guide](https://github.com/wildlifeai/wildlife-watcher-user-guide):** Comprehensive documentation and troubleshooting for the Wildlife Watcher system.
* **[API Services](https://github.com/wildlifeai/wildlife-api):** Dedicated API layer for secure cross-service communication.

---

## 🌊 Spyfish (Kūtai me te Ika)
A collaborative project with the University of Auckland focusing on marine monitoring, benthic habitat analysis, and citizen science.

* **[Kūtai me te Ika Core](https://github.com/wildlifeai/spyfish-kutai-me-te-ika):** The central repository for data processing and marine species monitoring scripts.
* **[Autotag Toolkit](https://github.com/wildlifeai/spyfish-autotag-toolkit):** Tools designed to automate and clean annotations generated by citizen scientists.
* **[Analysis Tools](https://github.com/wildlifeai/spyfish-analysis):** Research-specific scripts for processing biological data from the South Island.

---

## 🛠️ Hardware & Firmware
The physical "eyes and ears" on the ground. We believe in open hardware for transparent conservation.

* **[WW Hardware](https://github.com/wildlifeai/ww-hardware):** Open-source schematics, PCB designs, and mechanical CAD files for the Wildlife Watcher camera.
* **[Grove Vision AI Module](https://github.com/wildlifeai/Seeed_Grove_Vision_AI_Module_V2):** Firmware and configuration for Seeed Grove Vision AI modules used in real-time detection.
* **[Sensorsat WT Extensions](https://github.com/wildlifeai/Sensorsat_WT_Extensions_firmware):** Specialized firmware for expansion modules and satellite/LoRaWAN sensor integration.

---

## 🔬 AI Models & Research
Specific machine learning models trained for high-accuracy species identification across different environments.

* **[Bird ID Model](https://github.com/wildlifeai/bird-identification-model):** Computer vision models trained to identify endemic and rural bird species.
* **[Fish Detection Example](https://github.com/wildlifeai/fish_detection_example):** Reference models and training pipelines for underwater object detection and tracking.
* **[Mel Data Tools](https://github.com/wildlifeai/mel_data_tools):** Utilities for processing acoustic spectrograms for bioacoustic monitoring.

---

## 🏗️ Infrastructure & Support
The underlying systems that keep our organization and development teams running smoothly.

* **[.github](https://github.com/wildlifeai/.github):** The "Source of Truth" for Wildlife.ai. Contains our global [Contributing Guidelines](../CONTRIBUTING.md) and PR workflows.
* **[Open Accounting](https://github.com/wildlifeai/open_accounting):** Transparent financial tracking scripts for our charitable operations.
* **[Wildlife Containers](https://github.com/wildlifeai/wildlife-containers):** Docker configurations to ensure consistent development environments across the team.
